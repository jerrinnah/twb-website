# Brand assets

Place the brand logo here as **`logo.png`** (this exact filename):

```
assets/logo.png
```

Recommendations:
- **Transparent background PNG** (so it sits cleanly on light and dark sections).
- Roughly **400–600px wide**, with the wordmark trimmed close to the edges.
- The black wordmark is automatically rendered in **white** on dark backgrounds
  (footer, admin sidebar) via a CSS `invert` filter — so one black logo works
  everywhere. If you'd prefer a hand-made white version, save it as
  `logo-white.png` and tell me; I'll wire dark areas to use it instead.

Until `logo.png` is added, the site shows the text "The Walking Billboard" in its
place (the image `alt` text), so nothing looks broken.
